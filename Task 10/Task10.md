## Task 10

You are required to create an Ansible playbook to configure NGINX web server on Centos OS/AlmaLinux server.

Then deploy the NGINX configuration file and BASH script that you have created in Task 8 and Task 9 into the same server.

Then, you are required to create a bitbucket-pipelines.yml file that can be used to automate the deployment.

For more information about Bitbucket Pipelines, please refer to this URL: https://www.atlassian.com/software/bitbucket/features/pipelines
