## Task 6

Please use either Python, NodeJS or BASH to write a script that will process the data.

There is a data.csv file in the /root/ directory containing information about your customers.

It has the following columns:
```ID,NAME,CITY,COUNTRY,EMAIL,EMPNO,CONTRACTS,TOTALPAID```

Where
```
ID: Unique id of the customer
NAME: Official customer company name
CITY: Location city name
COUNTRY: Location country name
EMAIL: Email of the customer company contact person
EMPNO: Customer company employee number
CONTRACTS: Number of contracts signed with the customer
TOTALPAID: Total amount of money paid by customer
```

Read and analyse the data.csv file and output the answer to the following question:

- Which country has the largest number of customers’ contracts signed in it? 
- How many contracts does it have? If there is a tie and several countries have the same number of contracts, return all the affected countries' names.

The answer should be formatted as
```
The country that has the largest number of customers’ contracts: THAILAND (49400 contracts)
```
